MY PROJECTS

MERN BLOG

FULL STACK WEB APPLICATION

A MERN blog is a full-stack web application built using the MERN stack, which consists of four key technologies: MongoDB (for the database), Express.js (for the backend framework), React (for the frontend framework), and Node.js (for the server environment). In a MERN blog, users can create, read, update, and delete blog posts. The application allows for user registration and authentication, enabling users to have personalized profiles and the ability to comment on blog posts. It typically includes features such as real-time updates, search functionality, and a responsive and user-friendly interface.

PIZZA TIME

FULL STACK WEB APPLICATION

"Pizza Time" is a feature-rich e-commerce platform for a fictional restaurant, offering a diverse menu that includes pizza, sushi, and pasta. Developed using React and Laravel, it delivers a seamless user experience with dynamic components, spatial awareness through Leaflet Maps, pagination for efficient menu exploration, and smooth animations powered by Framer Motion. The use of UUID ensures robust data management, while React Lazy Load optimizes performance. MySQL2 provides a reliable database backend, and Express.js streamlines the ordering process, allowing for item customization and secure payment handling. With user registration, login, and profile management, "Pizza Time" enhances personalization, and Google reCAPTCHA ensures secure communication. It's a fusion of culinary delights and cutting-edge technology, making online food ordering a convenient and enjoyable journey for users.

PC BUILD

FULL STACK WEB APPLICATION

The "PC Build" full-stack web application, powered by PHP Laravel on the backend and Bootstrap on the frontend, simplifies the process of customizing and configuring personal computers. Users can effortlessly select and customize computer components while the application's MySQL database manages user profiles, PC configurations, and order details. With real-time price estimation, compatibility checks, and secure order processing, "PC Build" offers an intuitive and efficient platform for users to create and order their ideal PC setups.

E-COMMERCE

FULL STACK WEB APPLICATION

The "E-Commerce" full-stack web application leverages PHP Laravel for the backend and utilizes XAMPP with PHPMyAdmin for the MySQL database management. This platform offers a comprehensive online shopping experience, enabling users to browse, search, and purchase a wide range of products. The application seamlessly handles user registration, authentication, and order processing. It allows users to view product details, add items to their shopping cart, securely complete transactions, and track order histories. With its robust database system and user-friendly interface, the "E-Commerce" web application provides an efficient and secure platform for online retail.

NETFLIX CLONE

FRONT-END

The "Netflix Clone" frontend is a dynamic web application crafted with HTML, CSS, Bootstrap, and JavaScript, meticulously designed to mirror the captivating user interface and design elegance of the renowned streaming giant, Netflix. This immersive platform offers users a visually enticing and highly responsive interface, empowering them to seamlessly navigate through an extensive catalog of movies and TV shows. With the inclusion of JavaScript, users can now enjoy an even more interactive experience, featuring functionalities like dynamic carousels for effortless content browsing, modal windows that reveal in-depth details upon title selection, interactive buttons for playing trailers and adding titles to personal watchlists, a real-time search bar that efficiently filters and displays results, and robust user authentication capabilities, enabling personalized recommendations and tracking watch histories. The "Netflix Clone" frontend not only emulates the visual allure of Netflix but also enhances the user journey with captivating interactivity, making it an ideal starting point for users to explore and adapt for their own web projects.
